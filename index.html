CSS

  • Enter your CSS notes here
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • The 3 ways to create CSS are inline, which is directly in HTML, internal and external
  • The CSS Box Model consists of boxes inside of eachother in the order of margin, border, padding and content from outside to center
